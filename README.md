# 🌐 Nutran: The Future of Transparent Systems

FUTRAN is a futuristic, eco-friendly, and efficient urban transportation solution designed to revolutionize urban commuting. This project presents a modern web interface that showcases the technology, benefits, and use cases of FUTRAN in a clean and responsive design.

## 📦 Clone & Setup

### 1. Clone the repository

git clone https://github.com/NutranEV/futrangroup.git

### 2. Install dependencies

npm install

### 3. Run the development server

npm run dev

### 4. Generate the production build

npm run build

### 5. Preview the Production Build

npm run preview
